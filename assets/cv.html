Research Experience
Visual Odometry for Ground Vehicles
Aug 2014 - Dec 2014
under Prof. KS Venkatesh, Computer Vision Lab, Department of Electrical Engineering, IIT-Kanpur
Two stereo appraoches were implemented and evaluated on KITTI odometry benchmark:
Jiang2014: Model Based ICP: 3D points triangulated from stereo data, inliers detected via the use Iterative Closest Point Algorithm that uses a 1-DOF motion
model for initial estimate. Ecient PnP Algorithm is then used on the selected inliers to obtain the nal rotation and translation.
Howard2008: Inlier detection using an assumption of scene rigidity. Problem reduced to nding the maximum clique in a graph, solved using a heuristic.
Levenberg-Marquardt used for minimizing the reprojection error on the selected inliers.
A monocular visual odometry approach was also implemented:
Utilizes Nisters ve point algorithm for essential matrix computation.
Estimation of Range Flow from RGBD data
May 2014 - July 2014
under Prof. KS Venkatesh, Computer Vision Lab, Department of Electrical Engineering, IIT-Kanpur
Designed and implemented two approaches for estimation of Range Flow from RGB-D data, one using Total Least Squares Solution, the other utilizing
Global Minimum Energy Solution.
A third approach, Physical RGBD Flow, attempting to capture the physical ow (instead of apparent ow) was also implemented.
The Physical RGBD Flow approach was evaluated against data generated using a highly accurate 6-DOF tracking device, the Personal Space Tracker (PST-110).
Captured RGBD Data from a Microsoft Kinect using OpenNI and OpenCV libraries, and did all the processing using C/C++.

Landmark-based Robotic Localisation from RGBD data
December 2013
under Mr. Arjun Bhasin, Research Associate at Mechatronics Lab, IIT-Kanpur
Geometric Triangulation was used to determine the pose (modelled as a set of random variables with Gaussian Distribution) of a robot, from information
obtained from noisy landmarks.
A Microsoft Kinect was used to identify the landmarks (using color histogram based models), CAMshift algorithm was used to track these landmarks,
and bearing measurements were calculated using the depth data.
An error model for the data obtained from a Kinect was used along with the Error Propagation Law to arrive at the uncertainty in the nal pose computed
using the Geometric Localisation Algorithm.
